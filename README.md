## Running the project

You can run the project using docker:

```bash
docker-compose up -d
```

or locally:

```bash
cp .env.template .env
npm install;
npm start;
```

## Running the project

Three scripts are provided for testing:

``npm run test:unit`` to run unit tests

``npm run test:integration`` to run integration tests

``npm run test:e2e`` to run e2e tests
